# 🏔️ Jaunpur Soul - Heritage Homestay Website

**Jaunpur Soul** ek premium aur minimalistic website hai jo Uttarakhand ke Jaunpur region mein sthit ek heritage homestay ko promote karti hai. Iska design "Luxury meets Nature" vibe par based hai.

## ✨ Features
* **Modern UI:** Tailwind CSS ka use karke ek clean aur aesthetic look.
* **Responsive Design:** Mobile, Tablet aur Desktop sabhi devices par perfect chalti hai.
* **Inquiry Form:** EmailJS integration ke saath direct email functionality.
* **Interactive Elements:** Smooth scrolling aur hover effects.
* **Developer Signature:** Stylish console log credit aur footer credits.

## 🛠️ Tech Stack
* **Frontend:** React.js (v19)
* **Styling:** Tailwind CSS (via CDN/PostCSS)
* **Email Service:** EmailJS
* **Icons:** Lucide React / FontAwesome
